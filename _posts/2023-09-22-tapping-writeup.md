For this challenge we are asked to decode the following string

```
.--. .. -.-. --- -.-. - ..-. { -- ----- .-. ... ...-- -.-. ----- -.. ...-- .---- ... ..-. ..- -. ..--- -.... ---.. ...-- ---.. ..--- ....- -.... .---- ----- } 
```

Looks awefully like Morse code, so we can just use an online decoder like: https://morsecode.world/international/translator.html
